Estudando listas

Tipos de Listas

A tag <ol> cria uma lista ordenada, assim colocando os itens em ordem.

A tag <li> adiciona um item para a lista.

  1. Jan
  2. Feb
  3. Mar
  4. Apr
  5. May
  6. Jun
  7. Jul
  8. Aug
  9. Sep
  10. Oct
  11. Nov
  12. Dec

A tag <ul> cria uma lista não ordenada, assim não importando sua sequência.

  • Peter
  • John
  • Lee
  • Jane
  • Samara
  • Miles
  • Nathan

podemos criar listas com formatos diferentes e começando de algum número específico, como também colocar listas dentro de outras.

  1. Refrigerantes
  2. Balões
  3. Bolo
  4. Dôces
    • brigadeiros
    • Bombons
    • Balas
  5. Sucos

Lista de Definições

A tag <dl> cria uma lista de definicões.

A tag <dt> adiciona um termo para uma lista de definições.

A tag <dd> cria uma descrição para termos em listas de definições.

Termo 1
Descrição do termo 1
Termo 2
Descrição do termo 2
Termo 3
